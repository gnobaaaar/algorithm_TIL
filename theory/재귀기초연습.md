숫자가 들어가 있는 리스트가 주어질때, 리스트의 합을 리턴하는 함수

random.sample(0-99까지 중에서 임의의 10개를 만들어서 10개의 값을 가지는 리스트 변수 생성)

```python
def sum_list(data):
  if len(data) == 1:
    return data[0]
  
  

import random
data = random.sample(range(100), 10)
print(sum_list(data))
```
