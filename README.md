### 알고리즘 연습법

1. 연습장과 펜
2. 문제 분석 후 간단한 경우부터 복잡합 경우까지 테스트 경우를 적으면서 생각
3. 간단한 알고리즘이 보인다면 구현할 알고리즘을 세부사항으로 나누고, 문장으로 적어본다
4. 코드화하기 위해 구조 및 변수 고민
5. 각 문장을 코드 레벨로 적어본다
6. 데이터 구조 및 변수가 문장에 따라 제대로 동작하는지 적어보고 연습장과 펜으로 검증
